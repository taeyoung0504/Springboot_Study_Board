# SpringBoot_Board
스프링부트 게시판
<br>
<br>

##  목적 
스프링부트 학습을 위한 간단한 CURD 게시판 만들기
<br>

##  사용 언어 및 환경
- JAVA
- SpringBoot
- JPA
- Mysql
- gradle

<br>

# 구현 기능
1. 게시물 작성
2. 게시물 목록(리스트) 
3. 게시물 상세보기
4. 게시물에 대한 답변 작성(답변 개수 함께 표시)
5. 파일 업로드 / 다운로드  <- 파일다운로드 시 원본 파일명으로 보이지 않아 해결 중
6. 페이징
7. 로그인/로그아웃
8. 답변 개수 표시
9. 회원가입
<br>
<br>
